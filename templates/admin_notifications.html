📱 Notificações WhatsApp

Configure alertas de vendas no seu celular

⚙️ Configuração de Notificações

✅ Sistema Ativo

Toda vez que uma venda for realizada, você receberá uma notificação no WhatsApp.

📋 Como Configurar:

1️⃣
Acesse CallMeBot

Vá para callmebot.com e clique em "Get API Key"

2️⃣
Envie a mensagem no WhatsApp

Abra o WhatsApp e envie "I allow callmebot to send me messages" para o número do bot

3️⃣
Copie seu API Key

O bot vai responder com seu API Key. Guarde ele!

4️⃣
Configure no Railway

Adicione as variáveis no painel do Railway (veja abaixo)

🔧 Variáveis de Ambiente (Railway)

ADMIN_WHATSAPP=5511999999999 CALLMEBOT_API_KEY=1234567890

💡 Substitua pelo seu número de WhatsApp (com DDD, sem +) e seu API Key

🧪 Testar Notificação

Você deve receber uma mensagem no WhatsApp em alguns segundos

🛒 *NOVA VENDA NØVRA!*

👤 Cliente: João Silva

💳 Pagamento: Pix
